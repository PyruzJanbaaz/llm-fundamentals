# LLM Fundamentals

A practical guide for developers who want to become AI Engineers. This repository covers large language models from fundamentals to advanced engineering, including architecture, prompt design, embeddings, agents, and production best practices. It is designed to help you build the knowledge and skills needed to design, deploy, and maintain AI-first systems.
